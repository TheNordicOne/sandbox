# Rich Text Editor

Rich Text Editor uses Tiptap editor with the ngx-tiptap wrapper

**Requires:**
- ngx-tiptap
- @tiptap/core
- @tiptap/extension-link
- @tiptap/extension-table
- @tiptap/extension-underline
- @tiptap/pm
- @tiptap/starter-kit


**Modifications needed for real usage**
- Use icons from the actual app
- Use proper button and input directives/components


**Notes**
This RTE uses the popover api through the popover component. The popover component itself uses modern CSS, which is not widely available as of Dec. 2025.
